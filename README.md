# Classification Algorithm Comparison

This project applies multiple classification algorithms to a dataset and compares their performance.

Algorithms Used:
- Logistic Regression
- Random forest classifier
- KNeighborsClassifier
- GaussianNB
- AdaBoostClassifier
- SVM

Libraries:
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

Goal:
To identify the best performing classification model based on accuracy.
